# Newline by Fifth Third

_A Modern, Embedded Payments Platform Built for Growth_

## About Newline

Newline is an **embedded payments API platform powered by Fifth Third Bank**. Our modern API design enables seamless integration of **payment, deposit, and lending solutions** into your products. Whether you’re building for fintech, e-commerce, or enterprise applications, Newline provides secure, scalable, and developer-friendly tools.

- [**Newline API Documentation**](https://developers.newline53.com/reference/getting_started_guide).

- [**Newline Developer Documentation**](https://developers.newline53.com/docs/welcome)

- [**Contact Us**](https://newline53.com/contact-us.html)

## Key Features

- **Payments API** – Initiate and manage transactions securely.

- **Deposits API** – Enable embedded deposit accounts.

- **Lending API** – Offer credit and financing options.

- **Webhook Support** – Real-time event notifications.
