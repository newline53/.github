If you encounter unexpected behavior or issues while using our projects:

1. **Open an Issue** on GitHub.
2. Include the following information to help us reproduce the problem:

### Required Details

- **Steps to reproduce** the issue (be as specific as possible)
- **Expected vs. actual behavior**
- **Environment details**, such as:
  - Language or framework version
  - Operating System + version
  - The agent or tool you're using (if applicable)
  - Any relevant logs, stack traces, or configuration snippets

Providing complete information allows our team to triage and resolve problems efficiently.
